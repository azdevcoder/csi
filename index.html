A IA decide: Resolve agora ou Entrega para quem sabe resolver.

Como o sistema irá funcionar?

  • Todos os chamados serão centralizados no Gateway via APIs e Integrações
  • O sistema irá enviar ao Agente IA todos os chamados, para classificação
  • O agente irá analisar o chamado, caso não seja possível a resolução imediata, será encaminhado para o setor responsável
  • Assim que o chamado for finalizado, com a solução, o sistema será alimentado com as informações pertinentes

Como a IA irá trabalhar?

  • Fará uma análise do chamado, e classificará o nível de competência exigido para resolução
  • Caso haja em sua base de conhecimento, uma solução viável, será aplicada imediatamente
  • Caso contrário, o agente irá identificar o setor responsável e o usuário mais capacitado e com menor demanda para a solução (Demanda essa que será calculada baseado nos chamados já atribuidos pelo agente)
  • O usuário receberá a sua demanda, e será calculado um prazo para entrega.
